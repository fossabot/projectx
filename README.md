# Database Project Shop

## Synopsis

Simple SQL Querys given over an API to the VUE frontend
using mySQL

## Features

Product Page, Edit Product

### Installation

- yarn install || npm install
- yarn prod || npm run prod
- yarn start || npm run start

### Commands

- Root Repository
  - yarn start
  - yarn prod
  - yarn watch
  - yarn watch-wp
- Vue Client
  - yarn build
  - yarn dev
  - yarn test
  - yarn start
